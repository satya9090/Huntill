Angular setup
-------------
1.Download and insatall nodeJs.

2.open cmd and run > npm install -g @angular/cli command,This command install the angular framework.

3.Then go to project folder,inside this open cmd and run >npm start command.(to start project)

war file creation
-----------------
4.Inside project folder run > ng build --base-href=/talent-poll-ui/
  
    -> After run this command a dist folder is created in the project.

    -> Inside dist talent-poll-ui folder is created ,

    -> Copy the talent-poll-ui folder and paste it inside 

	C:\Program Files(x86)\Apache Software Foundation\Tomcat 9.0\webapps folder and start the tomcat server.
